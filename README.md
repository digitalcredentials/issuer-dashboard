## Credential Dashboard

A nextjs app for managing Verifiable Credential issuance, revocation, and reporting.
Uses:

* [DCC Credential Store](https://github.com/digitalcredentials/cred-store)
* [DCC Signing Service](https://github.com/digitalcredentials/signing-service)
* [DCC Transaction Service](https://github.com/digitalcredentials/transaction-service)
* [DCC Workflow Coordinator](https://github.com/digitalcredentials/workflow-coordinator)

This is based off the starter template for the [Next.js App Router Course](https://nextjs.org/learn) on the Next.js Website.
